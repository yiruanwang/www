yarn常用命令

# yarn常用命令

yarn 升级指定版本 （例：升级到1.21.3版本）

yarn upgrade v1.21.1
1

# npm

yarn 安装/升级最新版本

npm install yarn@latest -g
1

查看 yarn历史版本

npm view yarn versions --json
1

# Homebrew

你可以通过Homebrew 包管理器 (opens new window)安装 Yarn，如果没有安装 Node.js 它也可以安装。

brew install yarn
1

如果您使用nvm (opens new window)或类似的东西，您应该排除安装 Node.js 以便使用 nvm 的 Node.js 版本。

brew install yarn --without-node
1

# MacPorts

您可以通过MacPorts (opens new window)安装 Yarn。 如果您未安装 Node.js，这也为您安装它。

sudo port install yarn
1

# yarn 升级 扩展包

yarn upgrade-interactive
1
