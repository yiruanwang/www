# 开始我们得白嫖之旅吧:laughing::laughing::laughing::laughing:

## 首先我们创建Docs文档仓库

``` shell
mkdir vitepress && cd vitepress
```

### 初始化项目

``` shell
yarn init
```

### 安装`VitePress`

``` shell
yarn add vitepress --dev
```

### 创建一个docs页面

``` shell
mddir docs
echo '# Hello VitePress' > docs/index.md
```
::: tip
（如果出现乱码，手动创建`index.md`文件）
:::
